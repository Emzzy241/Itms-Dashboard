# Intelligent Transaction Monitoring Dashboard
made by Mojiboye Emmanuel Oluwole

## The Front-end for the Event-Driven Intelligent Transaction Monitoring System

## Technologies Used
* React
* Vite
* Firebase
* lucide-react

## Description
This is the Front-end that displays all the generated results for the Intelligent Transaction Monitoring System. The application is able to show a list of flagged transactions and unflagged transactions based on varying V1-V28 values and can provide the Compliance Team of a Company more information about a particular transaction.

## Known Bugs
* As at most recent Deployment date, there are currently no Known bugs.

## Installation
* Clone the GitHub repository to your local machine
```sh
    git clone origin https://github.com/Emzzy241/Itms-Dashboard.git
```
* Install all dependencies and devDependencies by running 
```sh
    npm install
```

* Build and run the project in development stage by running
```sh
    npm run dev
```
